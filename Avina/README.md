# 📚 پروژه آخرین نگهبان

یک پلتفرم جامع آموزشی برای نوجوانان با قابلیت‌های:**

- 🎓 مدرسه آنلاین (دوره‌های تدریس شده)
- 🎮 بازی‌های آموزشی
- 🏪 فروشگاه محصولات فیزیکی و دیجیتال
- 👤 مدیریت پروفایل و دستاوردها
- 💰 سیستم کوین اقتصادی
- 📊 ردیابی پیشرفت

## 🏗️ ساختار پروژه

```
Avina/
├── Components/
│   ├── Layout/
│   │   ├── MainLayout.razor        # صفحه‌بندی اصلی
│   │   └── NavMenu.razor            # منوی ناویگیشن
│   └── Pages/
│       ├── Home.razor               # صفحه اول
│       ├── School.razor             # صفحه مدرسه
│       ├── Games.razor              # صفحه بازی‌ها
│       ├── Store.razor              # صفحه فروشگاه
│       └── Profile.razor            # صفحه پروفایل
├── Models/
│   ├── User.cs                      # مدل کاربر
│   ├── Course.cs                    # مدل دوره
│   └── Product.cs                   # مدل محصول
├── Services/
│   ├── UserService.cs               # سرویس مدیریت کاربران
│   └── CourseService.cs             # سرویس مدیریت دوره‌ها
├── wwwroot/
│   ├── css/
│   │   ├── app.css                  # استایل‌های اصلی
│   │   └── app-custom.css           # استایل‌های سفارشی
│   ├── js/
│   │   └── app.js                   # اسکریپت‌های جاوااسکریپت
│   └── images/
│       ├── courses/                 # تصاویر دوره‌ها
│       ├── profiles/                # تصاویر پروفایل
│       ├── products/                # تصاویر محصولات
│       └── hero/                    # تصاویر قهرمان
├── Program.cs                       # فایل اصلی برنامه
└── appsettings.json                 # تنظیمات برنامه
```

## 🚀 شروع به کار

### پیش‌نیازها
- .NET 10
- Visual Studio 2026
- Bootstrap 5

### نصب و اجرا

1. **کلون کردن پروژه:**
```bash
git clone [repository-url]
cd Avina
```

2. **بازگشایی پروژه:**
```bash
code .
```

3. **اجرای پروژه:**
```bash
dotnet run
```

4. **باز کردن در مرورگر:**
```
http://localhost:5000
```

## 📱 صفحات

### 1. **Home (صفحه اول)**
- قسمت قهرمان معرفی برنامه
- آمار آماری
- دوره‌های پیشنهادی

### 3. **Games (بازی‌ها)**
- لیست بازی‌های آموزشی

### 4. **Store (فروشگاه)**

### 5. **Profile (پروفایل)**

- فعالیت‌های اخیر
- تنظیمات

## 🔧 سرویس‌ها

### UserService
مدیریت اطلاعات کاربران:
- `GetUserByIdAsync(id)` - دریافت کاربر
- `GetCurrentUserAsync()` - دریافت کاربر جاری
- `UpdateUserAsync(user)` - به‌روزرسانی کاربر
- `LoginAsync(email, password)` - ورود
- `LogoutAsync()` - خروج

### CourseService
مدیریت دوره‌ها:
- `GetAllCoursesAsync()` - تمام دوره‌ها
- `GetCourseByIdAsync(id)` - دوره خاص
- `GetFeaturedCoursesAsync()` - دوره‌های پیشنهادی
- `GetEnrolledCoursesAsync(userId)` - دوره‌های ثبت‌نام‌شده
