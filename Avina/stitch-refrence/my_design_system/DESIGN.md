# Design System Documentation

## Theme Overview

This document outlines the core aesthetic and functional principles of our design system, reflecting the chosen theme settings.

## Color Palette

Our color palette is built for a `dark` mode experience, emphasizing clarity and visual comfort in low-light environments.

*   **Primary Color:** `#1275e2` - A vibrant and commanding blue, used for key interactive elements, calls to action, and primary branding.
*   **Secondary Color:** `#5f78a3` - A soft, supportive blue-grey, ideal for less prominent UI elements, data visualizations, and secondary information.
*   **Tertiary Color:** `#c55b00` - An expressive and distinct orange, reserved for accents, highlights, badges, or decorative elements that require special attention.
*   **Neutral Color:** `#74777f` - A balanced neutral grey, forming the foundation for backgrounds, surfaces, and typography, ensuring readability and visual hierarchy.

## Typography

The chosen typeface for our system is `Inter`, a highly legible and versatile font designed for screens.

*   **Headlines:** `Inter` - Provides a modern and clean aesthetic for all titles and headings, ensuring strong hierarchy.
*   **Body Text:** `Inter` - Ensures consistency and readability for all paragraphs and longer textual content.
*   **Labels:** `Inter` - Applied to form labels, buttons, and other interactive text elements for clarity and uniformity.

## Shape and Form

The system incorporates a `moderate` level of roundedness (value: `2`) for UI elements, striking a balance between modern softness and functional precision. This offers a friendly yet professional appearance.

## Spacing

Our design maintains a `normal` level of spacing (value: `2`) between and within UI elements. This standard spacing promotes a balanced layout, preventing clutter while ensuring an efficient use of screen real estate.
