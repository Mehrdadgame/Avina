## ✅ نیا Upload Page

### فیچرز:

1. **Cloud Upload Zone** ☁️
   - Drag & Drop support
   - File selection
   - Modern animation

2. **Content Type Selection** 🎬

   - Audio
   - Image
   - Document/Book

3. **Content Form** ✏️
   - Title input
   - Description textarea
   - Tags/Discoverability

4. **Access Type** 🔐
   - Premium Content (💎)
   - Public (🌐)
   - Unlisted (🔗)
   - Private (🔒)

5. **Price Section** 💰
   - Shows when Premium selected
   - Customizable price

6. **Preview Section** 👁️
   - Real-time preview
   - Shows title, description
   - Shows type and price

7. **Action Buttons**
   - Discard (🗑️)
   - Create Tip (✅)

3. Fills in title & description

6. Submits or discards
