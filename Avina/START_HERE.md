## 🚀 شروع کرنے کے لیے

### Step 1: تصاویر ڈالیں
```
wwwroot/images/
├── courses/
├── games/
├── products/
├── profiles/
└── hero/
```

### Step 2: چلائیں
```powershell
dotnet run
```

### Step 3: کھولیں
```
http://localhost:5000
```
